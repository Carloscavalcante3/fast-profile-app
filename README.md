## 📝 Descrição
Este é um protótipo de rede social simples desenvolvido em React Native. O aplicativo permite ao usuário inserir um nome, visualizar uma foto de perfil circular e alternar entre duas imagens diferentes.

## 🛠️ Funcionalidades
- **TextInput**: Entrada de texto com bordas arredondadas e placeholder.
- **Image**: Foto de perfil circular com `resizeMode='cover'`.
- **Button**: Ação de salvar com alerta nativo e troca de fotos.
- **Layout**: Uso de Flexbox centralizado e componentes responsivos.

## 🚀 Como Executar
1. Instale as dependências: `npm install`
2. Inicie o projeto: `npx expo start`
3. Escaneie o QR Code com o aplicativo **Expo Go** no seu celular.

## ⚙️ Tecnologias
- React Native
- Expo
- JavaScript (Hooks: useState)
